# AirWar
# Update version1.0
Game planes shoot
It's easy game.
Have 2 plane. Player control 1 plane by mouse and left click to shoot.
Player plane has 5 hearts. If player plane is shooted 5 times, game over.
If player shoot plane enemy 10 time, player win game.
